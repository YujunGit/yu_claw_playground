# AI 截图吐槽机

一个轻量网页 playground：上传一张截图，选择页面类型，再输入一句你想达成的目标，它会给出 3 条“又毒又建设性”的产品点评。

## 这版做了什么

- 支持本地上传截图预览
- 支持选择页面类型：落地页 / App / Dashboard / 海报 / 其他
- 根据场景生成 3 条吐槽 + 改进建议
- 适合做 UI review 热身、创意讨论、产品吐槽开场

## 打开方式

直接在浏览器打开：

```bash
xdg-open index.html
```

## 说明

- 当前版本是启发式原型，不做真实图像识别
- 重点在于快速产出有趣、可讨论、可执行的点评
- 如果后续要升级，可以接图片理解模型，变成真正的截图审稿工具
