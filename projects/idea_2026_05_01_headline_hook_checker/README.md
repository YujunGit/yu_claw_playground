# 标题吸引力体检器

一个轻量的静态网页 playground：输入一个标题、首页首屏 headline 或一句主文案，快速判断它到底是偏平、偏空，还是已经有点抓人，并给出 3 条更有钩子的改写建议。

## 它能做什么

输入标题后，这个工具会输出：

- 吸引力评分
- 命中的问题信号（太空、太泛、太长、没结果感等）
- 当前这句标题仍然可取的地方
- 3 条更有吸引力的改写建议

## 适合场景

- 首页 headline 初稿自检
- Landing page 首屏文案打磨
- 社媒帖标题优化
- 判断一句标题是不是“看起来没错，但没人会停下来”

## 当前版本特点

这是一个 **纯前端、固定启发式规则** 的版本：

- 不联网
- 不调用外部模型
- 不依赖后端
- 只根据输入文本和常见标题信号做快速判断与改写

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 输入一个标题
- 可选补充背景
- 点击“开始体检”

## 后续可演进方向

- 支持不同风格改写（克制 / 锋利 / 转化导向）
- 支持中英双语 headline 检查
