# 产品定位撞车检查器

一个轻量的静态网页 playground：输入一句产品定位语句，快速判断它是否过于同质化，并指出常见的撞车表达、模糊区和优先改进方向。

## 它能做什么

输入一句定位语句后，这个工具会输出：

- 撞车风险分
- 明显的公版撞车词
- 主要问题
- 仍然可取的部分
- 优先修改建议

## 适合场景

- 产品定位初稿自检
- 首页标题 / 简介打磨
- 冷启动阶段找差异化表达
- 判断一句话是不是“谁都能这么说”

## 当前版本特点

这是一个 **纯前端、固定启发式规则** 的版本：

- 不联网
- 不调用外部模型
- 不依赖行业数据库
- 只根据输入文本和常见公版表达做快速判断

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 输入一句产品定位语句
- 可选补充背景
- 点击“开始检查”

## 后续可演进方向

- 支持自动给出多版差异化改写
- 增加行业专属撞车词表
- 支持品牌语气风格选择
- 支持输出更短的 slogan 版本
