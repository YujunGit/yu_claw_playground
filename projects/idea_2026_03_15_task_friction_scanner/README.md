# 任务摩擦扫描仪

一个超轻量网页 playground，帮你把“我知道要做，但就是起不来”的任务快速拆成可执行的第一步。

你只需要输入：

- 任务名称
- 预计时长
- 当前最卡的点（不知道从哪开始 / 太大了 / 容易分心 / 害怕做不好 / 需要沟通 / 缺资料）

它会立刻给出：

- 摩擦等级
- 阻力解释
- 一个 2 分钟可启动动作
- 一个 15 分钟冲刺版本
- 一个“如果还是不想做”的保底动作

## 打开方式

直接用浏览器打开：

```bash
xdg-open index.html
```

或者双击 `index.html`。

## 适合什么时候用

- 明明知道该干嘛，但总在启动前卡住
- 想把模糊任务压缩成一个可执行动作
- 需要快速找出“真正阻力”而不是继续空想

## 说明

这是一个启发式 playground，不是严肃的效率系统。
它的目标不是管理你，而是把阻力讲清楚，让你更容易立刻开始。
