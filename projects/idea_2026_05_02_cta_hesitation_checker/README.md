# CTA 犹豫指数检查器

一个轻量的静态网页 playground：输入一个按钮文案、CTA 区块或一句行动号召，快速判断它为什么“不够想点”，并给出更有行动感的改写建议。

## 它能做什么

输入 CTA 后，这个工具会输出：

- 犹豫指数评分
- 命中的问题信号（太泛、太弱、没结果感、没对象感等）
- 当前 CTA 还可取的地方
- 3 条更想让人点击/行动的改写建议

## 适合场景

- 首页 CTA 自检
- 按钮文案优化
- 落地页转化文案打磨
- 判断一句行动号召是不是“看了也懒得点”

## 当前版本特点

这是一个 **纯前端、固定启发式规则** 的版本：

- 不联网
- 不调用外部模型
- 不依赖后端
- 只根据输入文本和常见 CTA 信号做快速判断与改写

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 输入一句 CTA
- 可选补充场景背景
- 点击“开始检查”
