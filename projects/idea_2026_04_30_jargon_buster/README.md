# 介绍文案去黑话器

一个轻量的静态网页 playground：输入一段产品介绍或功能描述，快速识别里面的黑话、空话和抽象表达，并给出更像真人会说的话的改写建议。

## 它能做什么

输入文案后，这个工具会输出：

- 黑话密度评分
- 命中的黑话/抽象词
- 用户可能听不懂的地方
- 一版更直白的人话改写
- 可继续加强的具体化建议

## 适合场景

- 首页简介自检
- README / landing page 文案打磨
- 产品定位早期迭代
- 判断一句介绍是不是“听起来厉害但其实没说清楚”

## 当前版本特点

这是一个 **纯前端、固定启发式规则** 的版本：

- 不联网
- 不调用外部模型
- 不依赖后端
- 只根据输入文本和常见黑话表达做快速分析与改写

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 粘贴一段介绍文案
- 可选补一点产品背景
- 点击“开始翻译成人话”

## 后续可演进方向

- 支持多种语气风格改写
- 支持行业黑话词库切换
- 支持输出更短的 slogan 版本
- 支持对比“原文 vs 改写版”的可读性差异
