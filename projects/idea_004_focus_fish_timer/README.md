# Idea 004: Focus Fish Timer 🐟

## Concept
A simple terminal-based Pomodoro timer that features a friendly ASCII fish. Focus while the fish "swims" through time!

## How to use
Run the timer with the default 25 minutes:
```bash
python focus_fish.py
```

Or specify a custom focus duration in minutes:
```bash
python focus_fish.py --mins 10
```

## Features
- Terminal-friendly Pomodoro timer.
- Dynamic ASCII fish animation.
- Celebration screen upon completion.
- Focus-oriented interface.
