# 网页冷启动体检器

一个超轻量的静态网页 playground：输入一个网址或 landing page 描述，快速得到一份启发式的冷启动诊断。

## 它能做什么

基于你输入的文本，生成 3 个部分：

1. **第一印象**
2. **转化阻碍**
3. **可立即优化的 3 处**

同时给出 3 个简单分数：
- 清晰度
- 转化驱动力
- 信任感

## 注意

这不是自动抓网页的审计器，也不是 AI 实时分析服务。

当前版本是一个 **纯前端、可解释、固定规则** 的 playground：
- 不联网抓取网页
- 不读取页面真实 DOM
- 只根据你输入的 URL / 文案描述做启发式判断

所以它适合：
- 冷启动过一遍 landing page 思路
- 快速发现“有没有 CTA / 有没有证明 / 有没有说清楚给谁用”
- 做产品文案的第一轮自检

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 粘贴一个网址
- 粘贴一段产品介绍 / hero 文案 / 页面描述
- 或者两者一起给

然后点击 **开始体检**。

## 当前启发式规则（简化版）

会检测一些明显信号，例如：
- 是否有目标用户描述
- 是否有结果导向表达
- 是否有 CTA（如“立即体验 / 免费试用 / 预约 demo”）
- 是否有价格、案例、数据、客户评价等信任信号
- 是否存在功能堆叠过多的问题
- 是否提到使用方式或路径
