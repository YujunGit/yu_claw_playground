# 网页情绪温度计

一个超轻量网页小工具：

- 粘贴一段网页正文、广告文案、社媒长帖或介绍页文字
- 快速判断它更偏向 **冷静解释 / 营销洗脑 / 情绪裹挟**
- 给出命中信号和阅读建议，帮你更快识别“被推着走”的文风

## 打开方式

直接用浏览器打开：

```bash
xdg-open index.html
```

或者双击 `index.html`。

## 适合什么时候用

- 看到一段特别会“煽”的文案，想判断它到底在讲事实还是在推你下单
- 粗略过筛 landing page、销售页、公众号长文、社媒长帖
- 写作时反查：自己的文案是不是已经太用力了

## 说明

这是一个 **启发式 playground**，不是严肃 NLP 分类器。
它依赖规则和关键词做快速判断，目标是帮你形成更敏锐的文本直觉。
