# 一句话生成微型产品页

一个超轻量的静态网页 playground：输入一句产品想法，快速得到一版可继续手改的微型 landing page 草稿。

## 它能做什么

输入一句想法后，会自动展开出：

- hero 标题
- 副标题
- 主 CTA / 次 CTA
- 3 个卖点
- 一组信任信号
- 一版简化页面骨架

适合：
- 点子刚冒出来时快速整理
- 想先做个能给人看的首页第一稿
- 产品描述太散，需要先压成一页表达

## 当前版本特点

这是一个 **纯前端、可解释、固定规则** 的 playground：

- 不联网
- 不调用外部模型
- 不依赖后端
- 根据输入中的关键词和语义线索，拼出一版可用草稿

所以它不是“真正 AI 写作器”，而是一个适合快速起稿的创意脚手架。

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 输入一句产品想法
- 选择语气（清晰直接 / 温柔可信 / 有点锋利）
- 可选填写产品名
- 点击“生成页面”

## 示例

输入：

> 一个帮独立开发者把更新日志自动改写成像人写的一页式发版公告工具

会生成一版偏 landing page 风格的首页草稿，方便继续手动改文案。
