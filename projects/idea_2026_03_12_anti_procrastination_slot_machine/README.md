# 反拖延老虎机

一个超轻量网页小工具：

- 点一下，随机给你一个 **15 分钟以内的小任务**
- 再附一条开工文案
- 适合脑子发木、任务太大、不想开始的时候

## 打开方式

直接用浏览器打开：

```bash
xdg-open index.html
```

或者双击 `index.html`。

## MVP / v2 功能

- 随机任务池
- 随机嘴替文案
- 再抽一次
- 复制当前结果
- 最近抽取历史
- 任务分类筛选（清理 / 沟通 / 创作 / 推进）
- 嘴替风格切换（温柔 / 欠揍 / 冷酷 / 搞笑）
- 短时间内尽量避免重复抽到同一任务

## 设计原则

不是严肃效率系统，也不是完整待办工具。

它只是一个目标很小但很实用的东西：
**帮你在卡住时，立刻开始一个足够小的动作。**
