# Terminal Aquarium

A relaxing, interactive ASCII aquarium that lives in your terminal.

## Features
- **Animated Fish**: Multiple fish swim around the screen.
- **Feeding**: Press `f` to drop food into the tank. Fish will eat it if they are close enough.
- **Hunger & Mood**: Fish become hungry over time (indicated by a color change in Day mode). Feeding them restores their mood.
- **Day/Night Cycle**: Press `n` to toggle between Day and Night modes.
- **Responsive**: Adjusts to your terminal window size.

## How to Run
```bash
python aquarium.py
```

## Controls
- `f`: Feed the fish.
- `n`: Toggle Day/Night mode.
- `q`: Quit the aquarium.
