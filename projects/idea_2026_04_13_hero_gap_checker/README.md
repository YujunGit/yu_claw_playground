# 首页首屏信息缺口检查器

一个轻量的静态网页 playground：输入产品首页第一屏文案，快速判断这段文案是否讲清楚了对象、结果、动作、证据和基本路径。

## 它能做什么

输入一段首屏文案后，这个工具会输出：

- 清晰度分数
- 动作驱动分数
- 信任感分数
- 一组信息缺口检查
- 优先补强建议

## 适合场景

- landing page 初稿自检
- 首屏文案打磨
- 发给朋友或用户前先自己快速体检
- 找出“为什么看着还行，但就是不够打动人”的原因

## 当前版本特点

这是一个 **纯前端、固定启发式规则** 的版本：

- 不联网
- 不调用外部模型
- 不抓取真实网页
- 只根据输入文本做首屏信息完整度检查

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 粘贴首页标题 / 副标题 / CTA
- 可选补充产品背景
- 点击“开始检查”
