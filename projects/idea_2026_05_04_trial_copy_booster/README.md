# 免费试用文案补强器

一个轻量的静态网页 playground：输入一段 trial / demo / 免费体验相关文案，快速判断它为什么还不能降低用户的心理门槛，并给出更能让人愿意点进去试试的改写建议。

## 它能做什么

输入文案后，这个工具会输出：

- 试用门槛感评分
- 命中的问题信号（太空、太泛、没结果感、没风险消除等）
- 当前文案仍然可取的地方
- 3 条更能降低顾虑的改写建议

## 适合场景

- 免费试用按钮旁文案自检
- demo / trial 区块优化
- 降低注册前犹豫感
- 判断一句试用文案是不是“看了还是不敢点”

## 当前版本特点

这是一个 **纯前端、固定启发式规则** 的版本：

- 不联网
- 不调用外部模型
- 不依赖后端
- 只根据输入文本和常见试用文案信号做快速判断与改写

## 使用方式

直接用浏览器打开 `index.html`。

你可以：
- 输入一句试用相关文案
- 可选补充用户对象 / 使用场景
- 点击“开始检查”
